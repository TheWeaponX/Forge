# Objetivo

Este módulo é responsável por receber um novo usuário e preparar toda a plataforma para oferecer uma experiência personalizada.

O objetivo não é apenas cadastrar um usuário.

O objetivo é compreender quem ele é.

---

# Princípios

O cadastro deve ser:

• rápido

• intuitivo

• agradável

• personalizável

• opcionalmente completo

O usuário nunca deve sentir que está preenchendo um formulário enorme.

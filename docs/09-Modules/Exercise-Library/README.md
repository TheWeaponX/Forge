# 1. Objetivo

A Exercise Library é a base de conhecimento do Forge.

Ela não existe apenas para listar exercícios.

Ela ensina.

Explica.

Relaciona.

Recomenda.

Toda funcionalidade do Forge utiliza esta biblioteca.

# 2. Filosofia

Cada exercício deve responder:

• Como fazer?

• Por que fazer?

• Quais músculos trabalha?

• Quando utilizar?

• Quando evitar?

• Como evoluir?

• Quais erros evitar?

• Quais exercícios se relacionam?

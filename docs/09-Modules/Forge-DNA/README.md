# 1. O que é o Forge DNA

O Forge DNA é o núcleo de personalização da plataforma.

Ele representa o usuário através de um perfil vivo, construído a partir de seus dados, hábitos, treinos e evolução.

O objetivo não é rotular pessoas.

O objetivo é compreender padrões e utilizá-los para oferecer recomendações mais úteis.

O Forge DNA evolui continuamente.

---

# 2. Objetivos

• Personalizar o aplicativo.

• Melhorar recomendações.

• Ajudar a IA.

• Sugerir treinos.

# 3. Filosofia

O Forge DNA nunca define quem o usuário é.

Ele apenas descreve como ele está neste momento.

Toda característica pode evoluir.

# 9. Recomendações

O Forge DNA gera recomendações como:

• próximo treino;

• skill recomendada;

• exercício complementar;

• tempo de descanso;

• desafio ideal;

• grupo sugerido;

• conteúdo educativo.

---

# 10. Evolução

O perfil nunca é permanente.

Ele é recalculado periodicamente.

Toda alteração significativa gera uma nova versão do Forge DNA.

O usuário poderá visualizar sua evolução ao longo do tempo.

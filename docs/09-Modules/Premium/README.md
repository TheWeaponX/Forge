# 1. Visão Geral

O Premium Module gerencia as assinaturas do Forge.

Seu objetivo é oferecer funcionalidades avançadas sem comprometer a excelente experiência da versão gratuita.

---

# 2. Objetivos

- Financiar o desenvolvimento do Forge.
- Oferecer recursos avançados.
- Não limitar funcionalidades essenciais.
- Garantir sustentabilidade do projeto.

---

# 3. Filosofia

O Forge deve ser extremamente útil na versão gratuita.

O Premium acelera a evolução do usuário, mas nunca torna a versão gratuita inutilizável.

Não haverá anúncios na experiência principal.

# 14. Casos de Erro

- pagamento recusado;
- renovação falhou;
- compra pendente;
- conflito de licença;
- restauração incompleta.

---

# 15. Critérios

- sincronização entre dispositivos;
- desbloqueio imediato dos recursos;
- restauração simples;
- transparência na cobrança.

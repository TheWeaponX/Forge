# 1. Visão Geral

O Rankings Module organiza classificações entre usuários, amigos, grupos e guildas.

Seu objetivo é incentivar evolução saudável e não competição excessiva.

---

# 2. Objetivos

- incentivar consistência;
- celebrar evolução;
- reconhecer dedicação;
- criar eventos competitivos;
- fortalecer a comunidade.

---

# 3. Filosofia

O melhor ranking não é o que mostra apenas o primeiro colocado.

É o que mostra para cada usuário onde ele evoluiu.

Os rankings do Forge sempre priorizam progresso relativo, consistência e participação.
